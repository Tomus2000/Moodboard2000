# Assets Directory

This directory contains static assets for the Student Moodmeter app.

## Beach Image

For the best experience, add a beach-themed background image named `beach.jpg` to this directory.

### Recommended Specifications:
- Format: JPG or PNG
- Dimensions: 1920x1080 or larger (will be scaled)
- Theme: Calm, beach-themed (sandy beaches, ocean, sunset, etc.)
- Style: Soothing, calming colors (blues, teals, sand colors)

### Example Sources:
- Unsplash: https://unsplash.com/s/photos/beach
- Pexels: https://www.pexels.com/search/beach/

### Alternative:
If you don't have a beach image, the app will use a gradient background as a fallback.

## Logo (Optional)

You can also add a logo image named `logo.png` to this directory if you want to customize the app's branding.
